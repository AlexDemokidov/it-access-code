# Код доступа в IT — React + Docker

Сервис из двух страниц:

- `/` — ввод кода
- `/hero/:code` — результат с карточкой IT-героя

## Запуск

```bash
docker compose up --build
```

Открыть:

```text
http://localhost:8080
```

## Где менять героев и логику

Файл:

```text
src/main.jsx
```

- `heroes` — 10 карточек героев.
- `pickHero(code)` — логика выбора героя по коду.
- картинки героев лежат в `public/heroes`.


## Финальная правка

Первый экран не показывает список героев и не спойлерит количество персонажей. Герой появляется только на странице результата `/hero/:code`.
